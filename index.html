Como a conta funciona:
1) Custo base = filamento + embalagem + demais custos + (horas extras × R$/h).
2) Presencial: preço = custo base × multiplicador.
3) Marketplace: preço anunciado ajustado para sobrar o mesmo líquido após taxas.
4) Lucro real = líquido recebido − custo base.
